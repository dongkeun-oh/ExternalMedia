Detailed Description

ExternalThermodynamicState property struct.

The ExternalThermodynamicState propery struct defines all the properties that are computed by external Modelica medium models extending from PartialExternalTwoPhaseMedium. Note: the exported interface functions use typeless void *state instead of ExternalThermodynamicState *state as Modelica does not treat external struct types so far.
